## ğŸ¯ CORE CONCEPT

AgentBridge is a two-sided platform that bridges consumer AI agents with business service agents, enabling autonomous commerce through:
- **Consumer Side (B2C):** Personal AI agents that can autonomously discover, negotiate, and purchase services
- **Business Side (B2B):** Service provider agents that can accept, negotiate, and fulfill requests from consumer agents
- **Bridge Protocol:** Uses Unibase AIP 2.0 (ERC-8004 identity + x402 payments + decentralized memory) on BNB Chain

```typescript
// Use Unibase's identity contracts
import { IdentityRegistry } from '@unibase/aip-sdk';

// Create agent identity
const identity = await IdentityRegistry.createIdentity({
  owner: walletAddress,
  metadata: ipfsHash,
  agentType: 'consumer' | 'business'
});
```

**2. x402 Payment Protocol:**
```typescript
// Import x402 from Unibase SDK
import { X402Payment } from '@unibase/aip-sdk';

// Create payment channel
const payment = await X402Payment.createChannel({
  from: consumerAgentId,
  to: businessAgentId,
  amount: negotiatedPrice,
  escrowContract: escrowAddress
});
```

**3. Decentralized Memory:**
```typescript
// Use Unibase memory storage
import { MemoryStore } from '@unibase/aip-sdk';

// Store agent preferences
await MemoryStore.set({
  agentId: agentIdentity,
  key: 'preferences.hotel.maxPrice',
  value: '200',
  encrypted: true
});

// Retrieve memory
const prefs = await MemoryStore.get({
  agentId: agentIdentity,
  key: 'preferences.hotel.*'
});
```
